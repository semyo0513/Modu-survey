# SurveyMine 배포 가이드

## 1. Google Apps Script 배포

### 1단계: Google Sheets 생성
1. [Google Sheets](https://sheets.google.com) 에서 새 스프레드시트 생성
2. 이름: `SurveyMine DB`

### 2단계: Apps Script 프로젝트 연결
1. 스프레드시트 메뉴 → **확장 프로그램 → Apps Script**
2. `Code.gs`, `Schema.gs`, `Auth.gs`, `SurveyService.gs`, `ResponseService.gs` 파일을 각각 생성하고 내용을 붙여넣기

### 3단계: Web App 배포
1. **배포 → 새 배포** 클릭
2. 유형: `웹 앱`
3. 설정:
   - 실행 계정: **나 (본인 계정)**
   - 액세스 권한: **모든 사용자 (익명 포함)**
4. **배포** 클릭 → 웹 앱 URL 복사

### 4단계: API URL 설정
`js/api.js` 파일의 다음 줄을 수정:
```javascript
const WEB_APP_URL = '여기에_복사한_URL_붙여넣기';
```

## 2. GitHub Pages 배포

### 방법 A: GitHub Desktop 사용
1. GitHub에서 새 Repository 생성 (예: `surveymine`)
2. 이 폴더의 파일들을 Repository에 Push
3. Repository → **Settings → Pages**
4. Source: `main` 브랜치 / `/ (root)` 선택
5. **Save** → 배포 URL 확인

### 방법 B: Git CLI
```bash
git init
git add .
git commit -m "SurveyMine v2.0 초기 배포"
git remote add origin https://github.com/USERNAME/surveymine.git
git push -u origin main
```
그 후 GitHub Settings → Pages에서 활성화

## 3. 학교 데이터 초기화
GAS 배포 후 **Sheets → Schools 시트**에 학교 데이터를 추가:

| schoolId | schoolName | region | createdAt |
|----------|-----------|--------|-----------|
| SCH-001 | 경남중학교 | 경상남도 | 2026-07-07 |
| SCH-002 | 경남고등학교 | 경상남도 | 2026-07-07 |

또는 관리자 API (`schools.add`)를 통해 추가 가능.

## 4. QR 코드 배포 (응답자 직접 링크)

특정 설문의 QR 코드 생성 URL:
```
https://USERNAME.github.io/surveymine/respond.html?surveyId=SRV-XXXXXX
```

학교를 미리 지정하는 URL:
```
https://USERNAME.github.io/surveymine/?school=경남중학교
```
